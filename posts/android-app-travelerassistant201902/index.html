設定時間按鈕點下去後會出現 選完可以在預期時間發送

原理是透過AlertReceiver.class中的==BroadcastReceiver==

 1public class AlertReceiver extends BroadcastReceiver {
 2    @Override
 3    public void onReceive(Context context, Intent intent) {
 4        if(intent.getAction().equals("Hello"))
 5        {
 6            NotificationHelper notificationHelper = new NotificationHelper(context);
 7            NotificationCompat.Builder nb = notificationHelper.getChannelNotification(intent.getStringExtra("information"));
