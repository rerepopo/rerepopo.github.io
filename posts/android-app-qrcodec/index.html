Android App QRcodec

2019-03-02 4 分鐘閱讀 Android

偽目標：

  1. 製作QR code
  2. 解析QR code (1). 來自相機 (2). 來自網路

在Android中想要使用QRcode 應該都會套用ZXing套件

1    compile 'com.google.zxing:core:3.2.1'
2    compile 'com.journeyapps:zxing-android-embedded:3.2.0@aar'

BarcodeEncoder

透過hints可以設定自我修正程度與邊框大小 BarcodeEncoder會return Bitmap 這時候再放到ImageView即可

 1    public void updateImageAndContent(String text, BarcodeFormat barcodeFormat)
 2    {//ref: http://justimchung.blogspot.com/2017/10/android-qr-code.html
 3        ImageView ivCode = (ImageView) findViewById(R.id.ivCode);
 4        BarcodeEncoder encoder = new BarcodeEncoder();
 5        try {
 6            Map hints = new EnumMap(EncodeHintType.class);
 7            //hints.put(EncodeHintType.ERROR_CORRECTION, ErrorCorrectionLevel.H);
 8            hints.put(EncodeHintType.MARGIN, 2);
 9            Bitmap bit = encoder.encodeBitmap(text, barcodeFormat,
10                    500, 500,hints);
11            ivCode.setImageBitmap(bit);
12            deCodeBitmapToContent(bit);// check answer
13        } catch (WriterException e) {
14            e.printStackTrace();
15        }
16    }

解析QR code (下載網路圖片)

deCodeBitmapToContent

從網路照片解析的話 需要使用deCodeBitmapToContent這個方法 傳入==Bitmap== decode方法是result = reader.decode(binaryBitmap); 文字結果會放在String text = result.getText(); barcodeFormat會放在BarcodeFormat barcodeFormat = result.getBarcodeFormat();

 1private void deCodeBitmapToContent(Bitmap generatedQRCode)
 2    {//ref: https://stackoverflow.com/questions/32120988/qr-code-decoding-images-using-zxing-android
 3        if (generatedQRCode==null)
 4        {Toast.makeText(this, "At deCode, generatedQRCode==null", Toast.LENGTH_SHORT).show();
 5            return;
 6        }
 7        int width = generatedQRCode.getWidth();
 8        int height = generatedQRCode.getHeight();
 9        int[] pixels = new int[width * height];
10        generatedQRCode.getPixels(pixels, 0, width, 0, 0, width, height);
11
12        RGBLuminanceSource source = new RGBLuminanceSource(width, height, pixels);
13
14        BinaryBitmap binaryBitmap = new BinaryBitmap(new HybridBinarizer(source));
15
16        Reader reader = new MultiFormatReader();
17        Result result = null;
18        try {
19            result = reader.decode(binaryBitmap);
20        } catch (NotFoundException e) {
21            e.printStackTrace();
22        } catch (ChecksumException e) {
23            e.printStackTrace();
24        } catch (FormatException e) {
25            e.printStackTrace();
26        }
27        String text = result.getText();
28        BarcodeFormat barcodeFormat = result.getBarcodeFormat();
29        TextView textViewQRCode = (TextView) findViewById(R.id.decode_text);
30        textViewQRCode.setText(" barcodeFormat: " +barcodeFormat.toString()+"\n CONTENT: " + text);
31
32    }

DownloadImageTask

把要更新的圖片位置與網址字串傳到DownloadImageTask[ref]

1                EditText url = (EditText)findViewById(R.id.internet_scan_edittext);

3                Log.d("dddd",url_str);
4                new DownloadImageTask((ImageView) findViewById(R.id.ivCode)).execute(url_str);

這是一個AsyncTask，可以在背景下載

 1private class DownloadImageTask extends AsyncTask<String, Void, Bitmap> {
 2        //ref: http://finalfrank.pixnet.net/blog/post/43369915-%5Bandroid-%E9%96%8B%E7%99%BC%5D-%E5%B0%87%E5%A4%96%E9%83%A8%E7%B6%B2%E5%9D%80%E9%A1%AF%E7%A4%BA%E6%96%BC-imageview-%E4%B8%AD
 3        ImageView bmImage;
 4
 5        public DownloadImageTask(ImageView bmImage) {
 6            this.bmImage = bmImage;
 7        }
 8
 9        protected Bitmap doInBackground(String... urls) {
10            String urldisplay = urls[0];
11            Bitmap mIcon11 = null;
12            try {
13                InputStream in = new java.net.URL(urldisplay).openStream();
14                mIcon11 = BitmapFactory.decodeStream(in);
15            } catch (Exception e) {
16                Log.e("Error", e.getMessage());
17                e.printStackTrace();
18            }
19            return mIcon11;
20        }
21
22        protected void onPostExecute(Bitmap result) {
23            bmImage.setImageBitmap(result);
24            deCodeBitmapToContent(result);// check answer
25        }
26    }

解析QR code (開啟相機)

用一個按鈕發動相機 加一個onActivityResult收來自相機的返回資訊 結果放在 String text = result.getContents(); String barcodeFormat_str = result.getFormatName();

 1        scan_btn = (Button)findViewById(R.id.scan_btn);
 2        final Activity activity = this;
 3        scan_btn.setOnClickListener(new View.OnClickListener()
 4        {//ref: http://dangerlover9403.pixnet.net/blog/post/200742339-%5B%E6%95%99%E5%AD%B8%5D-qrcode-scanner(android-version)
 5         //ref: http://oldgrayduck.blogspot.com/2018/01/android-studio-zxing-qrcode.html
 6            //自製掃描圖示教學ref: https://www.jianshu.com/p/ef2ef751bf5b
 7
 8            @Override
 9            public void onClick(View view)
10            {
11                IntentIntegrator integrator = new IntentIntegrator(activity);
12                integrator.setPrompt("掃描中...");
13                integrator.setCameraId(0);//0:後  1:前鏡頭
14                integrator.setBeepEnabled(false);
15                integrator.setBarcodeImageEnabled(false);//该方法用于设置“被扫描的二维码图片”可以保存在本地//result.getBarcodeImagePath()
16                integrator.setOrientationLocked(false);//<!--如果要直立螢幕 要加這段/ -->调整手机方向时，扫描布局也会重新布置
17                integrator.setDesiredBarcodeFormats(IntentIntegrator.ALL_CODE_TYPES);//QR_CODE_TYPES
18                integrator.initiateScan();
19
20                //按下按鈕後 取消鍵盤顯示 看起來順一點
21                //ref: https://blog.csdn.net/ccpat/article/details/46717573
22                InputMethodManager imm = (InputMethodManager) getSystemService(Context.INPUT_METHOD_SERVICE);
23                if (imm != null) {
24                    imm.hideSoftInputFromWindow(getWindow().getDecorView().getWindowToken(), 0);
25                }
26
27            }
28        });
